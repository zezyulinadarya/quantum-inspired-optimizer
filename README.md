# quantum-inspired-optimizer
## 🎯 Постановка задачи

### Цель

Разработать и реализовать высокоэффективный **квантово-вдохновлённый алгоритм** для поиска основного состояния (состояния с минимальной энергией) в задачах модели Изинга.

### Контекст

Модель Изинга — это краеугольный камень статистической физики, который также является основой для формулировки широкого класса задач дискретной оптимизации. Современные квантовые и гибридные алгоритмы, такие как **QAOA**, **VQE** и алгоритмы **квантового отжига**, демонстрируют большой потенциал в решении таких задач.

### Формулировка задачи

Необходимо создать оптимизатор, который для заданной матрицы взаимодействий \( J \) и вектора внешних полей \( h \) находит конфигурацию спинов \( s \in \{-1, +1\}^n \), минимизирующую **гамильтониан Изинга**:

$$
H(s) = \sum_{i,j} J_{ij} s_i s_j + \sum_i h_i s_i
$$

**Ключевое требование:** использование **квантово-вдохновлённых подходов**.

### Входные данные: задача MaxCut

Для запуска предоставляются матрицы задачи **MaxCut** в постановке **QUBO** (Quadratic Unconstrained Binary Optimization) для заданного в файле графа. Это классический бенчмарк для бинарных алгоритмов.

**Суть задачи MaxCut:**
- Дан неориентированный граф
- Необходимо разбить вершины на две группы
- **Цель:** максимизировать количество рёбер, соединяющих вершины из разных групп (разрез)

### Связь с QUBO

Задача MaxCut сводится к минимизации квадратичной формы:

$$
E(x) = x^T Q x, \quad x \in \{0,1\}^n
$$

где \( Q \) — симметричная матрица, построенная по графу.

## Метрика качества

В качестве метрики качества рассматривается среднее отклонение от минимальной (эталонной) энергии:

$$
\text{Score} = \frac{1}{K} \sum_{k=1}^{K} \left( E_{\text{solution}}^{(k)} - E_{\text{min}}^{(k)} \right)
$$
